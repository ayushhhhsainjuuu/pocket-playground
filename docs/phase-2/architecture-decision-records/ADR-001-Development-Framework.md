\# ADR-001: Development Framework



\## Status

Approved



\## Context

Pocket Playground is a hub-based Android application that launches multiple small games and interactive tools.

The app must support mobile APIs and hardware features such as sensors.



The team has experience with JavaScript and React.



\## Decision

The team has selected React Native as the development framework.



\## Rationale

\- Strong support for Android devices

\- Large developer community

\- Easy integration of mobile APIs and sensors

\- Modular structure supports multiple mini-games

\- Fits within course scope



\## Consequences



\### Positive

\- Faster development due to familiarity

\- Easy navigation integration

\- Good performance for simple games



\### Negative

\- Some native features may require additional configuration
