\## Highlights and Accomplishments



During Phase 2, the team successfully:



\- Determined the development framework (React Native)

\- Defined navigation strategy (Hub + Stack navigation structure)

\- Selected hardware features (motion sensors, vibration, audio output)

\- Chose local data storage solution

\- Created structured Architecture Decision Records (ADR-001 to ADR-004)

\- Organized GitHub repository and documentation structure



All architectural decisions were documented clearly and reviewed for alignment with Phase 1 project goals.



---



\## Challenges and Issues



One of the main challenges was selecting hardware features that balanced creativity with feasibility.  

The team initially considered GPS and advanced biometric features, but these were determined to be outside the practical scope of the project timeline.



Another challenge was structuring the navigation in a way that supports multiple mini-games without overcomplicating the app structure.



These challenges were resolved through scope refinement and focusing on modular, scalable design decisions.
